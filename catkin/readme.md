# Aspen - Catkin

## Getting Started

1. Install `protoc` and `swift-protobuf`
```
$ brew install protobuf@3
$ brew install swift-protobuf
````

2. Generate the swift protobuf files
```
$ pwd
# ~/path/to/aspen/catkin

$ mkdir protobuf/
$ ./scripts/protoc.sh
```

4. Open the project with XCode and everything should be set!
